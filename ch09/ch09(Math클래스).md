# Math클래스
- 수학관련 static메서드의 집합
```java
Math.abc() // 주어진 값의 절대값을 반환한다 (양수->음수, 음수->양수)
Math.cell() // 주어진 값을 올림하여 반환한다
Math.floor() // 주어진 값을 버림하면 반환한다
Math.max() // 두 값을 비교하여 큰 값을 반환한다
Math.min() // 두 값을 비교하여 작은 값을 반환한다
Math.random() // 0.0~1.0 사이의 임의의 double값을 반환한다(1.0은 범위에 포함되어있지 않다)
Math.rint() //반올림인데 짝수 반올림이라는데....
Math.round() // 일반적인 반올림 반환
```
